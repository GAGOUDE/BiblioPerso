📖 BiblioPerso – Suivi de lecture biblique personnalisé
BiblioPerso est une application web intuitive qui permet aux chrétiens de suivre leur progression de lecture de la Bible, chapitre par chapitre, livre par livre.

✨ Fonctionnalités principales
Deux versions bibliques disponibles : vous pouvez choisir entre la Bible évangélique (66 livres) et la Bible catholique (73 livres). L'application adapte automatiquement la liste des livres et les objectifs.

Suivi précis : chaque chapitre peut être marqué comme "lu" d'un simple clic. La progression est sauvegardée automatiquement dans votre navigateur.

Sélection multiple : maintenez la touche Ctrl (ou Cmd sur Mac) et glissez pour sélectionner et valider plusieurs chapitres d'un coup.

Validation rapide : un bouton "Tout valider" permet de marquer un livre entier comme complété.

Statistiques claires : visualisez en un coup d'œil le nombre de livres lus dans l'Ancien Testament, le Nouveau Testament, et le nombre de Bibles complètes. Des graphiques en barres et en camembert illustrent votre progression.

Filtres pratiques : affichez uniquement les livres de l'Ancien ou du Nouveau Testament, ou utilisez la recherche pour trouver un livre spécifique.

Mode clair / sombre : basculez entre les deux thèmes selon votre préférence, avec sauvegarde automatique.

Interface responsive : sur mobile et tablette, un menu hamburger permet d'accéder facilement aux différentes sections.

🎯 Pour qui ?
Chrétiens évangéliques (Bible LSG 66 livres)

Chrétiens catholiques (Bible 73 livres incluant les deutérocanoniques)

Toute personne souhaitant suivre sa lecture biblique de manière organisée

💾 Données personnelles
Toutes vos données sont stockées localement sur votre appareil (localStorage). Aucune information n'est envoyée sur un serveur. Vous êtes seul maître de votre progression.
